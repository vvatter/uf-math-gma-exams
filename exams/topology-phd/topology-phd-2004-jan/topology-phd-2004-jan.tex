\begin{examinstructions}
Work the following problems and show all work. Support all statements to the best of your ability.
\end{examinstructions}
\begin{examproblems}{0}
\item
Show that no covering of the 2-torus has the homotopy type of a figure eight.
\item
Show that there is no retraction of the Möbius band onto its boundary.
\item
Does there exist a covering space of the figure eight that has a non-trivial abelian fundamental group?
\item
Show that the 2-torus with a point deleted does not admit a structure of a topological group.
\item
Show that the space \(R^2 \setminus A\) is path connected for any countable set~\(A\).
\end{examproblems}
\begin{examinstructions}
Answer the following with complete definitions or statements or short proofs.
\end{examinstructions}
\begin{examproblems}{5}
\item
State the Seifert–van Kampen Theorem.
\item
State the Lefschetz Fixed Point Theorem.
\item
State the Baire Category Theorem.
\item
State the Arzelà–Ascoli Theorem.
\item
Let~\(X\) be a finite complex. Find \(\chi(X \times S^n)\).
\item
State the Five Lemma.
\item
State the Poincaré Duality Theorem.
\item
Define retraction and deformation retraction.
\item
State the Mayer–Vietoris exact sequence for cohomology.
\item
Let~\(X\) be a complete metric space such that no point is isolated. Can~\(X\) be countable?
\end{examproblems}
