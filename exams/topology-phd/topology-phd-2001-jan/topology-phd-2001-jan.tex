\begin{examinstructions}
Work the following problems and show all work. Support all statements to the best of your ability.
\end{examinstructions}
\begin{examproblems}{0}
\item
Show that there is a continuous function from~\(I\) onto \(I^2\).
\item
State and prove the Brouwer Fixed Point Theorem. You may assume that \(H_n(S^n;\mathbb{Z})=\mathbb{Z}\) for all \(n\geq 1\).
\item
Show that \(H_n(S^n;\mathbb{Z})=\mathbb{Z}\) and that \(H_0(S^n;\mathbb{Z})=\mathbb{Z}\) for \(n\geq 1\).
\item
Show that the fundamental group of \(S^n\) is trivial for \(n>1\).
\item
Show that \(\pi_1(P_n)=\mathbb{Z}_2\) for \(n>1\), where \(P_n\) is~\(n\)-dimensional real projective space.
\end{examproblems}
\begin{examinstructions}
Answer the following with complete definitions or statements or short proofs.
\end{examinstructions}
\begin{examproblems}{5}
\item
State the Seifert–van Kampen Theorem.
\item
State the Hahn–Mazurkiewicz Theorem.
\item
State the Lefschetz Fixed Point Theorem for compact simplicial complexes.
\item
Define the cone of a space~\(X\). Define the suspension of~\(X\).
\item
State the exact homology sequence for a pair of spaces.
\item
State the Jordan Curve Theorem.
\item
State the Contraction Mapping Theorem.
\item
State the Ascoli (Arzelà–Ascoli) Theorem.
\item
Let~\(A\) and~\(B\) be disjoint closed sets in a metric space~\(X\). Show that there is a continuous function \(f:X\to[0,1]\) such that \(f^{-1}(1)=A\) and \(f^{-1}(0)=B\).
\item
Show that~\(I\) and \(I^2\) are not homeomorphic. Show that \(I^2\) and \(I^3\) are not homeomorphic.
\end{examproblems}
