# Logic, PhD exam, May 1993

*This examination is divided into 4 segments, with 3 questions in each segment. Problems 1–3 concern General Logic; problems 4–6 concern Model Theory; problems 7–9 concern Set Theory; and problems 10–12 concern Recursion Theory. Answer any 2 questions from each segment.*

**1.** State Gödel’s incompleteness theorem. Explain the importance of Gödel numbering.

**2.** State and prove the compactness theorem for Predicate logic. You may assume the completeness theorem.

**3.** Prove that if \(T\) is a countable theory with arbitrarily large finite models then \(T\) has an infinite model.

**4.** State and sketch the proof of the Łoś theorem on ultraproducts.

**5.** State and sketch a proof of the elementary substructure version of the downward Löwenheim–Skolem Theorem.

**6.** Prove that the theory of dense linear orders without endpoints is complete.

**7.** Prove that the well ordering principle implies the axiom of choice.

**8.** Prove, using the axioms of ZFC, that if \(X\) is any set then the transitive closure of \(X\) is also a set.

**9.** Show that a countable union of countable sets is countable, pointing out any use of the axiom of choice.

**10.** Show that a function \(\phi:\omega\longrightarrow\omega\) is (partial) recursive iff its graph is recursively enumerable, and that if \(\phi\) is total and recursive then its graph is recursive.

**11.** State the halting problem, and show that it is undecidable. You may use the fact that there is a recursive enumeration of the (partial) recursive functions.

**12.** Show that if \(A\ne\varnothing\) is the domain of a partial recursive function \(f\) then there is a total recursive function \(g\) such that \(A\) is the range of \(g\).
