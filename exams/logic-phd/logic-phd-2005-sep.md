# Logic, PhD exam, September 2005

*Answer six questions, including at least one from each of the following sections: Section 1 (Problems 1–3), Section 2 (Problems 4–6), and Section 3 (Problems 7–9).*

**1.** State and prove the compactness theorem for propositional logic.

**2.** Show that the theory of the class of infinite sets (in the language of equality) is complete and decidable, but not finitely axiomatizable.

**3.** State the Łoś Theorem for ultraproducts and sketch a proof.

**4.** Show that Zorn’s Lemma implies that any set can be well-ordered.

**5.** Sketch a proof that \(ZFC \cup \{\neg CH\}\) is consistent.

**6.** Show that any countable union of countable sets is countable.

**7.** Sketch a proof of Gödel’s incompleteness Theorem for Arithmetic.

**8.** Show that an infinite set \(X \subseteq \omega\) is the domain of a partial recursive function if and only if \(X\) is the range of \(\phi\) for some partial recursive function \(\phi\).

**9.** Show how to construct a simple recursively enumerable set \(A\) (meaning that \(\omega-A\) has no infinite r.e. subset).
