# Analysis, PhD exam, September 2010

*Write solutions in a neat and logical fashion, giving complete reasons for all steps.*

*Attempt EIGHT problems.*

**1.** Let \(f\in L^1[0,1]\). Suppose that
\[
\int_0^1 x^n f(x)\,dx=0 \quad\text{for all } n=0,1,2,\ldots
\]
Prove that \(f=0\) almost everywhere. (Hint: reduce the problem to showing that if \(\int_E f(x)\,dx=0\) for every Borel \(E\), then \(f=0\) a.e.)

**2.** Let \((X,\mathcal M,\mu)\) be a \(\sigma\)-finite measure space, and let \(g:X\to\mathbb C\) be a measurable function. Prove that the operator \(M_g:f\mapsto gf\) is bounded on \(L^p(\mu)\) (\(1\leq p\leq\infty\)) if and only if \(g\in L^\infty(\mu)\), in which case \(\lVert M_g\rVert=\lVert g\rVert_\infty\).

**3.** This problem has two parts.
* a) Define monotone class, and state the Monotone Class Theorem.
* b) State the Fubini–Tonelli theorem.

**4.** This problem has two parts.
* a) Let \(\nu\) be a signed measure on \((X,\mathcal M)\). Define what it means for a set to be positive, negative or null for \(\nu\), and state the Hahn decomposition theorem.
* b) Prove that if \(\nu\) is a signed measure, then there exist unique positive measures \(\nu^+,\nu^-\) so that \(\nu=\nu^+-\nu^-\) and \(\nu^+\perp\nu^-\).

**5.** Let \((k_n)_{n=1}^\infty\) be a sequence of measurable, \(1\)-periodic functions on \(\mathbb R\) with the following properties:

Prove that if \(f\in L^1[-\tfrac12,\tfrac12]\), then \(f*k_n\to f\) in \(L^1[-\tfrac12,\tfrac12]\).

(Here \(*\) denotes convolution on \([-\tfrac12,\tfrac12]\): \((f*g)(x)=\int_{-1/2}^{1/2}f(t)g(x-t)\,dt\).)
* a) \(k_n\geq 0\) a.e., for all \(n\).
* b) \(\displaystyle \int_{-1/2}^{1/2}k_n(t)\,dt=1\) for all \(n\).
* c) For each \(\delta>0\), \(k_n\to 0\) uniformly on the set \(\delta\leq |t|\leq\tfrac12\).
