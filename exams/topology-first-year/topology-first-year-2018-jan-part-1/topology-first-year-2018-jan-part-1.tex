\begin{examinstructions}
Answer the following problems and show all your work. Support all statements to the best of your ability.
\end{examinstructions}
\begin{examproblems}{0}
\item
A function \(f:\mathbb{N}\to\mathbb{Z}\) is called eventually zero if there is some~\(N\) such that \(f(n)=0\) for all \(n\geq N\). Show that the set of functions \(f:\mathbb{N}\to\mathbb{Z}\) that is eventually zero is countable.
\item
Let \(f:X\to Y\) be continuous and onto. Prove that if~\(X\) is compact then so is~\(Y\).
\item
Let \(f,g:X\to Y\) be continuous maps to a Hausdorff space. Prove that the set \(\{x\mid f(x)=g(x)\}\) is a closed subset of~\(X\).
\item
Prove that if~\(X\) is a compact Hausdorff space, then~\(X\) is a normal space.
\item
State and prove the Contraction Mapping Theorem.
\end{examproblems}
\begin{examinstructions}
Answer the following with complete definitions or statements or short proofs.
\end{examinstructions}
\begin{examproblems}{5}
\item
Prove that \(\{[a,b)\mid a<b\}\) is a basis for a topology on~\(\mathbb{R}\).
\item
Give the definition of a quotient map.
\item
Prove that there is no continuous map \(f:\mathbb{R}\to\{0,1\}\) that is surjective.
\item
Does a connected space need to be path connected?
\item
State the Intermediate Value Theorem.
\item
Describe the one-point compactification of \((0,1)\cup(2,3)\).
\item
State the Urysohn Lemma.
\item
State the Tietze Extension Theorem.
\item
Give an example of a space that is Hausdorff but not regular.
\item
Is \(\mathbb{R}^{\omega}\) connected in the uniform topology?
\end{examproblems}
