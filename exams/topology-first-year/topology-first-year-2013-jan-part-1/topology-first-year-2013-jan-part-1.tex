\begin{examinstructions}
Work the following problems and show all work. Support all statements to the best of your ability.
\end{examinstructions}
\begin{examproblems}{0}
\item
Show that the set of functions \(f:\mathbb{Z}\to\mathbb{Z}\) that are eventually constant is countable.
\item
Let~\(X\) be a connected metric space having more than one point. Can~\(X\) be countable?
\item
Show that every continuous map \(f:[0,1]\to[0,1]\) has a fixed point. Is this true for continuous maps
\begin{examsubparts}
\item[\textnormal{(a)}]
\(f:(0,1)\to(0,1)\)?
\item[\textnormal{(b)}]
\(f:[0,1]\to[0,1)\)?
\end{examsubparts}
\item
Let~\(A\) be a proper subset of~\(X\) and let~\(B\) be a proper subset of~\(Y\). If~\(X\) and~\(Y\) are connected, show that \((X\times Y)-(A\times B)\) is connected.
\item
Let \(f:X\to X\) be a map of a complete metric space to itself that satisfies the following condition: There exists \(\lambda<1\) such that \(d(f(x),f(y))\leq\lambda d(x,y)\). Prove that~\(f\) is continuous. Show that~\(f\) has a fixed point and the fixed point is unique.
\end{examproblems}
\begin{examinstructions}
Answer the following with complete definitions or statements or short proofs.
\end{examinstructions}
\begin{examproblems}{5}
\item
State the Intermediate Value Theorem.
\item
Does there exist a continuous surjective map from the 2-sphere \(S^2\) to the interval \((0,1)\)?
\item
Does there exist a continuous injective map from the 2-sphere \(S^2\) to the interval \((0,1)\)?
\item
State the Cantor–Schröder–Bernstein Theorem.
\item
Is every connected space path connected?
\item
What is a basis of a topology? Does the set of all half-open intervals \(\{(a,b], [b,c) \mid a<b, c<d\}\) form a basis of a topology on~\(\mathbb{R}\)?
\item
State the Extreme Value Theorem.
\item
Is \(\mathbb{R}^{\omega}\) connected in the uniform topology?
\item
Is the subspace \(X=\{(x,y)\in\mathbb{R}^2\mid y=\frac{1}{x}\}\) compact? locally compact?
\item
Describe all connected subsets of~\(\mathbb{R}\).
\end{examproblems}
