# Serious Mathematical Errors and Ambiguities

These are apparent mathematical errors, missing hypotheses, undefined symbols, or ambiguous formulas in the source exams. The extraction preserves the source reading because no correction was uniquely determined. This report contains **36 open review records across 32 exams**.

The links use jsDelivr’s GitHub CDN and point to the files on the `main` branch.
